## 使用建议

1. **基础阶段**：

   - 先熟悉"c++整理.md"中的基础知识点
   - 尝试完成"C++模拟试题-1-2.pdf"，巩固基础概念

2. **期中复习**：

   - 系统学习"c++期中复习资料"
   - 自测知识掌握程度

3. **期末准备**：

   - 完成所有模拟题
   - 分析历年真题，找出重点和难点
   - 重点复习往年考试中的高频考点

4. **查漏补缺**：
   - 通过答案解析找出自己的薄弱环节
   - 针对性地复习相关章节

## 重点内容

C++相比 C 语言的主要新增特性：

1. **面向对象编程**

   - 类和对象
   - 封装、继承和多态
   - 构造函数和析构函数

2. **高级特性**

   - 函数重载
   - 运算符重载
   - 模板（函数模板和类模板）
   - 异常处理

3. **标准模板库(STL)**
   - 容器（vector、list、map 等）
   - 迭代器
   - 算法
   - 函数对象

这些资料适合已经学习 C 语言并开始学习 C++的学生，帮助理解 C++的新特性以及面向对象编程的概念。
